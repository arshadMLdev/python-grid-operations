# 🧮 Python Grid Operations

A Python program to create and perform operations on a grid (matrix), including search, row sums, and uniqueness check.

---

## 🚀 Features

* Create a grid dynamically (user input)
* Search for a number in the grid
* Calculate sum of each row
* Check whether all elements are unique

---

## 🧠 Concepts Used

* Nested loops
* Lists and 2D arrays
* Functions (modular programming)
* User input handling

---

## ▶️ How to Run

1. Run the Python file:

   ```bash
   python main.py
   ```
2. Enter grid values when prompted
3. Enter a number to search

---

## 📌 Example

Grid:
1 2 3
4 5 6
7 8 9

Output:

* Number found in grid
* Sum of each row
* All numbers are unique
